HTML

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page.

  • HTML is the standard markup language for creating Web pages.
  • HTML (HyperText Markup Language) is the code that is used to structure a web page and its content.
  • There are four required tags in HTML. These are, html, title, head and body


HTML Formatting Elements

  • strong text
  • bold text
  • emphasized text
  • italic text
  • small text
  • mark the .text
  • Delete the text
  • Subscript text and Superscript text

About HTML :

The first version of HTML was written by Tim Berners-Lee in 1993. Since then, there have been many different versions of HTML. The most widely used version throughout the 2000's was HTML 4.01, which became an official standard in December 1999.
